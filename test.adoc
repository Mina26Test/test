== Overview
This is an *AsciiDoc* file used for testing. AsciiDoc is more powerful than Markdown and is often used for technical documentation and books.

== Features
* **Extensible**: Supports includes and variables.
* **Native TOC**: Can generate a Table of Contents easily.
* **Rich Admonitions**: Supports Notes, Tips, and Warnings.

---

[source,python]
----
def greet(name):
    print(f"Hello, {name}!")

greet("World")
----

[NOTE]
====
AsciiDoc is superior for large-scale documentation because it handles cross-references and complex tables better than Markdown.
